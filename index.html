Acerca de PAPI LAUNCHER

Un fork con identidad propia

PAPI LAUNCHER es un launcher de Minecraft de código abierto (GPL v3) basado en Hello Minecraft! Launcher (HMCL) creado por huangyuhui. Mantenido por endersan17, este fork se enfoca en:

🎨 Interfaz moderna con 6 temas neón personalizables
📦 Sistema de contenedores para organizar mods, resource packs, shaders y mundos
🚀 Modo offline activado por defecto (sin registros)
🔄 Actualizaciones automáticas desde GitHub
🎮 Discord Rich Presence integrado

Características

Todo lo que necesitas en un solo launcher

PAPI LAUNCHER incluye todo lo esencial para una experiencia Minecraft completa, sin publicidad, sin telemetría y con total transparencia.

🎨

6 Temas Neón

Void Green, Blue Launcher, Cyan Pulse, Electric Purple, Magenta Glow y Acid Lime. Todos con efecto glassmorphism.

📦

Contenedores Inteligentes

Panel de 3 columnas, búsqueda con debounce, perfiles de lanzamiento por contenedor y validación automática de resource packs.

Modo Offline + authlib-injector

Modo offline activado por defecto. Soporte para authlib-injector y cuentas Microsoft próximamente.

Multi-plataforma

Windows (x86-64, ARM64), Linux (x86-64, ARM64, RISC-V, LoongArch64), macOS (x86-64, ARM64) y FreeBSD.

Multi-idioma

Inglés, Español, Chino, Japonés, Ruso, Ucraniano y Chino Clásico. Traducción completa de la interfaz.

Auto-Update + Discord RP

Arquitectura Técnica

Cómo está construido

1. HMCLBoot

Bootstrap loader (Java 8 compatible)

  • Carga el launcher principal
  • Verifica versión de Java (mínimo 11)
  • Muestra splash screen

2. HMCLCore

Librería central

  • Auth (Microsoft, offline, authlib-injector)
  • Descarga de versiones de Minecraft
  • Lanzamiento del juego
  • Gestión de mods (CurseForge, Modrinth)
  • Utilidades (IO, JSON, red)

3. HMCL

Interfaz de usuario (JavaFX)

  • UI completa con temas y contenedores
  • Sistema de actualización
  • Discord RPC
  • Personalización

Tecnologías

Dependencias principales

Kotlin + Java JavaFX + JFoenix Gradle (Kotlin DSL) Gson JNA jsoup nanohttpd Discord RPC OpenNBT JUnit 5

Especificaciones Técnicas

Requisitos y Build

⚙️ Requisitos Mínimos

  • Java: 11 (mínimo) / 17 (recomendado) / 21 (ideal)
  • Sistema: Windows, Linux, macOS o FreeBSD
  • Arquitectura: x86-64, ARM64, RISC-V, LoongArch64

📦 Build

  • Lenguaje: Kotlin + Java
  • Build: Gradle (Kotlin DSL)
  • UI: JavaFX + JFoenix
  • Output: papi-launcher-<version>.jar

FAQ

Preguntas frecuentes

¿Es gratis?

Sí, Papi Launcher es un proyecto de código abierto pensado para la comunidad.

¿Funciona en Windows?

Sí, el launcher está pensado para funcionar en Windows.

¿Puedo usar mods?

Sí, incluye compatibilidad con Forge, Fabric y NeoForge para gestionar experiencias con mods.

¿Necesito Minecraft Premium?

No, no es necesario tener una cuenta premium de Minecraft para usar el launcher, tristemente aun no puedes poner cuentas de microsoft ¡Pero estamos trabajando en ello!

Primeros Pasos

Comienza con PAPI LAUNCHER

1

Descarga

Descarga la última versión desde GitHub Releases — elige .jar, .exe o .sh según tu sistema operativo.

Crear una cuenta sin conexión

Crear una cuenta sin conexión

Sigue estos pasos para crear una cuenta offline y empezar a jugar sin necesidad de registrarte.

1

Accede a Cuentas

Haz clic en la sección Cuentas en el menú lateral izquierdo.

2

Selecciona Sin conexión

Selecciona Añadir cuenta y elige la opción Sin conexión.

3

Escribe tu nombre

Ingresa tu nombre de usuario (máximo 16 caracteres, solo letras, números y guiones bajos).

4

¡Listo!

Haz clic en Iniciar sesión y disfruta. Así se ve con un nombre válido.

Descargar Minecraft

Descargar Minecraft

Instala el juego en un solo clic

1

Ve a Descargar

Haz clic en Descargar en el menú lateral izquierdo.

2

Elige una versión

Busca la versión de Minecraft que quieres instalar. Todas las versiones están disponibles.

3

Descarga automática

Selecciona la versión y el launcher la descargará automáticamente. Una vez lista, haz clic en Jugar.

Instalar Mods

Añade mods fácilmente

PAPI LAUNCHER facilita la instalación de mods con soporte para Forge, Fabric y NeoForge.

🔧

Elige un Modloader

Selecciona Forge, Fabric o NeoForge al crear un contenedor o perfil. El launcher descargará el modloader automáticamente.

📂

Agrega tus Mods

Coloca los archivos .jar en la carpeta de mods de tu contenedor, o usa la opción "Agregar Mod" desde el launcher.

📥

Importa Modpacks

Soporta importación de modpacks desde CurseForge (.zip), Modrinth (.mrpack) y MultiMC.

5

¡Tema aplicado!

Sistema de Contenedores

Organiza tu contenido de Minecraft

Los contenedores agrupan mods, resource packs, shader packs y mundos para una experiencia de juego organizada.

+

Crear un Contenedor

Haz clic en "Crear Contenedor", asigna un nombre y descripción, selecciona la versión de Minecraft, agrega mods, resource packs, shaders o mundos.

📦

Gestión de Contenido

Mods, Resource Packs con validación automática de pack.mcmeta, Shader Packs y Mundos: todo organizado en un solo lugar.

⚙️

Perfiles de Lanzamiento

Cada contenedor puede tener su propio perfil con versión específica, argumentos JVM, resolución, memoria máxima/mínima e IP de servidor.
