Vista previa del Launcher

Así se ve PapiLauncher

Interfaz moderna, intuitiva y diseñada para que pases menos tiempo configurando y más tiempo jugando.

⚡ Inicio rápido 📦 Gestión de instancias 📂 Sistema de contenedores 🧩 Soporte para mods

Características

Control, privacidad y mods en un solo launcher.

Papi Launcher busca ofrecer la facilidad de uso de TLauncher, pero sin publicidad invasiva, sin telemetría innecesaria y con mayor transparencia gracias a su código abierto.

OS

Código Abierto

Código auditable y transparente para que la comunidad pueda revisar, mejorar y confiar.

PR

Enfoque en la Privacidad

Sin telemetría innecesaria y con decisiones pensadas para proteger tus datos.

AD

Sin Anuncios

Una experiencia limpia, directa y sin publicidad invasiva al iniciar tus partidas.

MC

Soporte para Cuentas de Minecraft

Compatible con cuentas Microsoft y cuentas locales, con mas opciones planificadas.

FX

Compatibilidad con Forge, Fabric y NeoForge

Gestiona versiones Vanilla y modloaders populares con base en OpenLauncher.

IN

Perfiles e Instancias Personalizadas

Separa configuraciones, mods y mundos para mantener cada experiencia ordenada.

FAQ

Preguntas frecuentes

¿Es gratis?

Sí, Papi Launcher es un proyecto de código abierto pensado para la comunidad.

¿Funciona en Windows?

Sí, el launcher está pensado para funcionar en Windows.

¿Puedo usar mods?

Sí, incluye compatibilidad con Forge, Fabric y NeoForge para gestionar experiencias con mods.

¿Necesito Minecraft Premium?

No, no es necesario tener una cuenta premium de Minecraft para usar el launcher, tristemente aun no puedes poner cuentas de microsoft ¡Pero estamos trabajando en ello!
